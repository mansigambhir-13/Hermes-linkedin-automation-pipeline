# Rehearsal — Sample Image-Generation Prompts (marktech-structured)

**10 complete, ready-to-test image prompts for real Rehearsal post concepts.**
Built on the Directive-04 structure (rich subject + structured sections + brand DNA + critical requisite).
Each is grounded in real Rehearsal facts (scoring vocab, briefs, modules) — not generic. Use these as the
Phase-2 probe test cases: run them through fal, judge on-topic + on-brand + clean text.

**Brand canvas applied to every prompt** (from the real assets): dark editorial field `#0a0a0a`,
Raleway type, the signature gradient `#9677f8→#4e44fd→#ff4859→#00c483` as ONE focal accent, generous
negative space, conceptual/editorial illustration (never literal stock photography).

> Poster prompts carry approved copy lines (the hook/punchline) + COPY SAFETY RULES so text renders clean.
> Editorial prompts forbid all text. Swap the `[Reference style description]` line for the real Gemini
> vision-grounding output once Upgrade C is wired; until then the inline descriptor approximates it.

## How to use these

- **As Directive-04 probe cases:** run 2-3 of the POSTER prompts (e.g. #1, #2, #6) through fal/Ideogram
  first — they test the hardest thing (clean in-image text + on-brand). Judge: on-topic? on-brand? text clean?
- The copy lines here are **drafts to test rendering**, not final captions — the real CTA/hashtags still
  come from `locked-config.json` and the agent's caption step; these prompts test the *image*, not the post copy.
- Swap each `[Reference style description]` for the real Gemini vision-grounding output (Upgrade C) once
  wired — that's the step that pushes "on-brand-ish" to "looks like Rehearsal."
- If poster text renders garbled even with Ideogram, that's the signal (per the stopping rule) to move text
  off-image to a typographic-card/template approach and keep AI generation for the editorial backgrounds only.
