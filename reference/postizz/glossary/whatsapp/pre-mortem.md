## Caption (copy-paste)

*Pre-Mortem*

Before the project starts, imagine it has *already failed*. Spectacularly. Now explain why.

Psychologist *Gary Klein* found this small reframe changes the room. Don't rock the boat becomes show how smart you are by spotting the crack. Treating the failure as certain surfaces far more reasons for it than politely asking what might go wrong.

The risks you name before you start are the only ones you can still avoid.
